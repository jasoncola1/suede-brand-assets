# Suede Brand Assets

Public brand and listing packet for Suede Labs, Johnny Suede (`SUEDE`), and the Suede programmable IP ecosystem.

## Brand Positioning

Suede Labs is a programmable IP and creator ownership platform for AI-native media.

Primary positioning:

```text
Programmable IP and creator ownership infrastructure for AI-native media.
```

Short description:

```text
Suede Labs helps creators register ownership, prove provenance, connect creative assets to licensing workflows, and expose media/IP actions to agent-ready commerce.
```

Token description:

```text
SUEDE is the utility token connected to Suede Labs, a programmable IP and creator ownership platform for AI-native media.
```

Category priority:

1. Programmable IP
2. Creator ownership
3. Creative RWA / media assets
4. Provenance and licensing
5. x402 / ACP agent commerce
6. AI music as a core media use case

## Logo Usage

- Use the approved logo on dark, light, or transparent backgrounds.
- Preserve clear space around the mark.
- Do not stretch, skew, recolor, crop, or add effects to the mark.
- Use PNG for listings and directories that require square transparent images.
- Use OpenGraph images for social cards, media previews, and public docs.
